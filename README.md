# kr-data
각종 데이터 링크 (내가 필요한 것 중심으로 모음)
## 교통
### [도로표지안내시스템](http://www.roadsign.go.kr)
- 도로 표지판의 위치, 사진 등
### [레일.블루](https://rail.blue)
- 다른 서비스들과 달리 개인이 운영함.  
  data.go.kr 자료나 TAGO 등 공공기관이 운영하는 서비스들은 대부분 데이터 갱신이 안되거나 활용하기 어렵게 돼있음.
- 열차 운전 정보, 노선 정보 등
- 노선 정보를 활용 사례: https://github.com/gnlow/kr.bns.json
